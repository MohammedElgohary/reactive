Multi-Style Binding

Three ways to bind multiple styles reactively: individual bindStyle(), object-based bindStyles(), and reactive style objects.

Method 1 — Individual bindStyle()
Bind each CSS property separately with its own reactive source.
bindStyle('#box1', 'background-color', () => state.backgroundColor);
bindStyle('#box1', 'color', () => state.color);
bindStyle('#box1', 'border-radius', () => state.borderRadius);
Method 2 — bindStyles() with functions
Pass a map of property → getter function in one call.
bindStyles('#box2', {
  'background-color': () => state.backgroundColor,
  'color': () => state.color,
  'transform': () => state.transform
});
Method 3 — bindStyles() with reactive object
Bind directly to a reactive style object — mutations auto-propagate.
const styleObject = reactive({ 'background-color': '#ff6b6b', ... });
bindStyles('#box3', styleObject);
