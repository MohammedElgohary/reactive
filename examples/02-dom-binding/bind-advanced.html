Advanced DOM Binding

bindAttr(), bindProp(), bindStyles(), bindMultiple(), and render() for complex scenarios.

1. bindAttr() — HTML attributes
Bind any HTML attribute reactively. Dangerous attributes (on*) and unsafe URLs are blocked automatically.
Visit Link 🔍 Hover to see title
disabled: false
href: https://example.com

2. bindProp() — DOM properties
Bind DOM properties (not attributes) — useful for value, checked, selectedIndex. Dangerous properties like innerHTML are blocked.
value: (empty)
checked: false
selectedIndex: 0
3. bindStyles() — multiple styles at once
Pass a record of style properties to bind them all in a single call — cleaner than multiple bindStyle() calls.

This paragraph has multiple reactive styles applied via bindStyles().

4. bindMultiple() — mixed bindings in one call
Apply text, class, style, and attribute bindings to multiple elements in a single declarative array.

5. render() — dynamic HTML lists
Render a reactive list as HTML. User-provided text is escaped; the template itself is trusted.
