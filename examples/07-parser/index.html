Write reactive UIs directly in HTML. Define state with reactive() — the library auto-mounts on DOMContentLoaded. Zero manual wiring.

Syntax reference
Text{{ counter.count }}
Attribute:disabled="counter.count === 0"
Class:class="{ active: state.on }"
Style:style="{ color: state.color }"
Show/hide:show="counter.count > 0"
HTML:html="state.richContent"
Two-way:model="user.name"
Event@click="counter.count++"
